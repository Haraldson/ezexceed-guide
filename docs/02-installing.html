Installing

Common

  1. Ensure dependencies are met:
    • eZ Publish 4.6 or higher
    • ezwebin extension
    • ezjscore extension
    • PHP 5.3
    • MySQL 5
    • Web server of your liking

  • eZ Publish version 4.7 or higher

    • 4.6 might work, but it's not a target
    • We are working on proper support for eZ Publish 5
    • ezjscore 1.3.0 or higher

    • ezwebin 1.8.0 or higher

    • Apache version 1.3 or 2.x in prefork mode

    • PHP version 5.3 or higher, 5.4 is recommended
    • Database server: MySQL 5 (UTF-8 is required)

  • Install extension into /my/ez/extension/ezexceed

     cd /my/ez/extension
 git clone git@github.com:KeyteqLabs/ezexceed.git
 cd ezexceed

  • Run the included SQL script to add eZ Exceed specifics to your database

     mysql -h host -u user -D my-database-name -p < sql/mysql/schema.sql

  • Activate the eZ Exceed extension in site.ini for every siteaccess where you want eZ Exceed enabled

     [ExtensionSettings]
 ActiveAccessExtensions[]=ezexceed

  • Add the eZ Exceed extension to the list of site designs for every siteaccess where you want eZ Exceed enabled

     [DesignSettings]
 AdditionalSiteDesignList[]=ezexceed
